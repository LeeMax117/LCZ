# 1.任务解析
## 目标
此次任务本意为使用卫星图根据城市的热岛效应，来对未来的城市规划做一个指导预测。
>### 阶段一
第一阶段，按大赛要求，需要使用两颗卫星的卫星图，以及相应的分类，对其进行分析、训练，得到一个有效的模型，从而可以对未知的卫星图进行准确的预测。
